# 🏛️ 中国传媒大学 侵权商品检测系统

自动搜索淘宝"中国传媒大学"相关商品，智能识别非书籍类侵权商品，进入详情页截图，生成 Excel 报告（含嵌入截图）。  
支持 **DeepSeek AI 侵权校验**，对商品进行精确到每一分的侵权评分。

> 💡 **无需任何 API Key 即可使用基础检测功能**，AI 校验为可选增强模块。

---

## ✨ 功能亮点

| 功能 | 说明 |
|------|------|
| 🚀 **一键检测** | 双击启动，自动搜索、翻页、识别、截图、生成报告 |
| 🧠 **五步核查** | 商标关键词 → 标识比对 → 商品类别 → 商家授权 → 混淆风险 |
| 🤖 **AI 侵权校验** | 接入 DeepSeek API，对商品进行精确到每一分的侵权评分 |
| 🖼️ **截图嵌入** | 每个侵权商品的详情页截图直接嵌入 Excel 报告 |
| 🔄 **自动翻页** | 支持多页扫描，随机跳页防反爬 |
| 🛡️ **反爬策略** | 模拟真人滚动、随机延迟、滑块验证处理、Cookie 持久化 |
| 🛑 **随时停止** | 检测过程中可随时终止，已收集结果保留并生成报告 |
| 🪟 **加载动画** | 启动时显示加载进度窗口，告别黑框等待 |

## 🚀 快速开始

### 1️⃣ 安装 Python

https://www.python.org/downloads/release/python-3913/

→ 找到 **"Windows installer (64-bit)"**  
→ **安装时务必勾选 "Add Python to PATH"**

### 2️⃣ 配置 DeepSeek API（可选）

打开 `backend/config.py`，填入你的 DeepSeek API Key：

```python
DEEPSEEK_API_KEY = "sk-你的key"
```

> 没有 Key 不影响基础检测功能，只是无法使用 AI 校验。

### 3️⃣ 双击运行

```
解压 cuc-trademark-detector 文件夹
双击 start.bat
```

**首次运行较慢**（约 1-2 分钟）：
1. 自动安装依赖包
2. 自动下载 Chromium 浏览器（约 50MB）
3. 弹出加载动画窗口 → 自动进入主界面

### 4️⃣ 开始检测

1. 点击 **"🚀 开始检测"**
2. 程序打开浏览器，自动搜索并逐页扫描商品
3. 自动识别侵权商品 → 进入详情页截图
4. 完成后自动生成 Excel 报告（截图直接嵌入表格）

### 5️⃣ AI 侵权校验（可选）

检测完成后，点击 **"🤖 AI侵权校验"** 按钮：
- 系统调用 DeepSeek API 对所有疑似侵权商品进行二次智能鉴别
- 分析商品标题、价格、商家等信息
- 给出侵权/非侵权判断及精确到每一分的置信度评分
- Excel 报告新增"AI侵权校验"、"AI置信度"、"AI建议"三列

### 6️⃣ 终止检测

检测过程中可随时点击 **"🛑 停止检测"**：
- 已收集到的结果会保留并生成报告
- 浏览器保持打开，可点击"重新检测"继续

## 🧠 侵权判定规则

### 五步核查法

| 步骤 | 检测项 | 说明 |
|------|--------|------|
| ① | **商标关键词** | 标题是否含"中国传媒大学/中传/CUC/校徽" |
| ② | **标识比对** | 是否含完整校名/简称/缩写 |
| ③ | **商品类别** | 是否属于服装、文具、饰品等侵权高发类别 |
| ④ | **商家授权** | 非北京商家推定无授权 |
| ⑤ | **混淆风险** | 标题是否含"纪念/周边/官方"等易混淆词 |

### AI 侵权校验（DeepSeek）

- **爬取中自动增强**：规则判定置信度低时自动调用 AI 辅助判断
- **爬取后批量校验**：点击"🤖 AI侵权校验"按钮，对全部结果进行二次鉴别
- AI 会分析商品标题、价格、商家等信息，给出侵权/非侵权判断及精确到每一分的置信度评分
- **核心铁律**：只要不是正规出版的书籍/教材，标题含"中国传媒大学"或"中传"或"CUC"且商品属于服装/文具/饰品/纪念品等周边类别的，一律从严判定为侵权

## 🛡️ 反爬策略

| 策略 | 说明 |
|------|------|
| ✅ 模拟真人滚动 | 每页上下滚动，行为更像真人 |
| ✅ 随机延迟 | 翻页随机延迟 2~5 秒 |
| ✅ 滑块验证处理 | 自动关闭 + 手动等待（30秒超时） |
| ✅ Cookie 持久化 | 登录凭证自动保存，避免重复登录 |
| ✅ 浏览器指纹伪装 | 隐藏自动化特征 |
