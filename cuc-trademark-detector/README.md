# 中国传媒大学 侵权商品检测系统

自动搜索淘宝"中国传媒大学"相关商品，识别非书籍类侵权商品，进入详情页截图，生成 Excel 报告（含嵌入截图）。  
支持 **DeepSeek AI 深度验证**（可选），对疑似侵权商品进行二次智能鉴别。

> 💡 **无需任何 API Key 即可使用基础检测功能**，AI 验证为可选增强模块。


## 环境要求

- **Windows 10/11**（推荐 Edge 浏览器）
- **Python 3.9+**

## 使用步骤

### 1️⃣ 装 Python
https://www.python.org/downloads/release/python-3913/
→ 找到 "Windows installer (64-bit)"
→ **安装时勾选 "Add Python to PATH"**

### 2️⃣ 配置 DeepSeek API（可选，用于AI验证）
打开 `backend/config.py`，填入你的 DeepSeek API Key：
```python
DEEPSEEK_API_KEY = "sk-你的key"
```
> 没有 Key 不影响基础检测功能，只是无法使用 AI 验证。

### 3️⃣ 双击运行
```
解压 cuc-trademark-detector 文件夹
双击 start.bat
```

**首次运行较慢**（约 1-2 分钟）：
1. 安装依赖包
2. 自动下载 Chromium 浏览器（约 50MB）
3. 启动检测窗口

### 4️⃣ 开始检测
- 点 **"🚀 开始检测"**
- 程序会打开浏览器，自动搜索并逐页扫描商品
- 逐个非书籍商品进入详情页截图
- 完成后自动生成 Excel 报告

### 5️⃣ AI深度验证（可选）
- 检测完成后，点击 **"🤖 AI深度验证"** 按钮
- 系统会调用 DeepSeek API 对所有疑似侵权商品进行二次智能鉴别
- 验证完成后可选择生成单独的 AI 验证报告

### 6️⃣ 终止检测（可选）
- 检测过程中可随时点击 **"🛑 终止检测"**
- 已收集到的结果会保留并生成报告
- 浏览器保持打开，可点击"重新检测"继续

## 功能说明

### Browser 支持（自动检测）
| 浏览器 | 效果 |
|--------|------|
| 🟢 Edge (推荐) | 使用本机已安装的Edge，兼容性最好 |
| 🟢 Chrome | 使用本机已安装的Chrome |
| 🟡 Chromium | 内置无头浏览器（兜底） |

### 侵权判定规则（五步核查）
1. **商标关键词** → 标题是否含"中国传媒大学/中传/CUC/校徽"
2. **标识比对** → 是否含完整校名/简称/缩写
3. **商品类别** → 是否属于服装、文具、饰品等侵权高发类别
4. **商家授权** → 非北京商家推定无授权
5. **混淆风险** → 标题是否含"纪念/周边/官方"等易混淆词

### AI 深度验证（DeepSeek）
- **爬取中自动增强**：规则判定置信度低时自动调用 AI 辅助判断
- **爬取后批量验证**：点击"🤖 AI深度验证"按钮，对全部结果进行二次鉴别
- AI 会分析商品标题、价格、商家等信息，给出侵权/非侵权判断及置信度

### 反爬策略
- ✅ 每页模拟上下滚动，行为更像真人
- ✅ 翻页随机延迟（2~5秒）
- ✅ 滑块验证自动关闭 + 手动等待（30秒超时）
- ✅ Cookie 持久化，避免重复登录

### 输出文件
- `data/reports/侵权检测报告_时间戳.xlsx` → 原始检测报告（商品截图直接嵌入表格，URL可点击跳转）
- `data/ai_reports/侵权检测报告_AI验证_时间戳.xlsx` → AI深度验证报告（单独目录，带时间戳方便对应）
- `data/screenshots/` → 每个商品的详情页截图
